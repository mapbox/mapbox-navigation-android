show

open fun show(alertText: String!, duration: Long, showLoading: Boolean): Unit

Animates the View from top down to its set position.

If a non-null duration is passed, a countdown loading bar will show to indicate the View will be automatically dismissed if not clicked.

Parameters

alertText - String!: text to be shown in the View

duration - Long: in milliseconds, how long the view will be shown

showLoading - Boolean: true if should show the progress bar, false if not
