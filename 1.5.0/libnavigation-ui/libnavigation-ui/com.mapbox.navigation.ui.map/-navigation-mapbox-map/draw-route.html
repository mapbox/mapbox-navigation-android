drawRoute

open fun drawRoute(@NonNull route: DirectionsRoute): Unit

Will draw the given DirectionsRoute on the map using the colors defined in your given style.

In most of the scenarios, this call can be wired to the RoutesObserver#onRoutesChanged(List) to display the route that MapboxNavigation uses to compute the route progress.

Parameters

route - DirectionsRoute: to be drawn
