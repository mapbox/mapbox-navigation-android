updateMapFpsThrottleEnabled

open fun updateMapFpsThrottleEnabled(isEnabled: Boolean): Unit

Enabled by default, the navigation map will throttle frames per second when the application has limited resources, such as when the device is running on battery power.

Throttling will also only take effect when the camera is currently tracking the user location.

Parameters

isEnabled - Boolean: true to enable (default), false to render at device ability
