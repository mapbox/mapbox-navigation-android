vanishingRouteLineUpdateIntervalNano

open fun vanishingRouteLineUpdateIntervalNano(vanishingRouteLineUpdateIntervalNano: Long): Builder!

Set minimum update interval (in nanoseconds) of the vanishing point when enabled.

Defaults to NavigationConstants#DEFAULT_VANISHING_POINT_MIN_UPDATE_INTERVAL_NANO. Decreasing this number will improve the smoothness of updates but impact the performance.

Parameters

vanishingRouteLineUpdateIntervalNano - Long: minimum update interval
