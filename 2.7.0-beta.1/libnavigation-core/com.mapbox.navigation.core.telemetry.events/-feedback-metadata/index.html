class FeedbackMetadata

Feedback metadata is used as part of MapboxNavigation.postUserFeedback to send deferred feedback. It contains data(like session ids, locations before and after call and so on) from a particular point of time/location when MapboxNavigation.provideFeedbackMetadataWrapper is called.

Note: MapboxNavigation.provideFeedbackMetadataWrapper returns wrapper of FeedbackMetadata that collect locations after call under the hood.
