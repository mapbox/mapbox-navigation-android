fun requestNavigationCameraToOverview(animatorListener: Animator.AnimatorListener? = null)
Content copied to clipboard

Executes a transition to OVERVIEW state. When started, goes to TRANSITION_TO_OVERVIEW and to the final OVERVIEW when ended. If transition is canceled, state goes to IDLE.

The target camera position is obtained with ViewportDataSource.getViewportData.

animatorListener

invoked on transition's progress
