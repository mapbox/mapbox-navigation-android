followingAnchorPropertyOverride

fun followingAnchorPropertyOverride(value: ScreenCoordinate?)
Content copied to clipboard

Whenever evaluate is called, the source produces ViewportData updates with opinionated values for all camera properties.

Use this method to override the Center Camera Property. As long as the override is present, it will be used for all ViewportData following updates instead of the opinionated value.
