setVanishingOffset

fun setVanishingOffset(offset: Double): RouteLineState.VanishingRouteLineUpdateState?
Content copied to clipboard

Sets the value of the vanishing point of the route line to the value specified. This is used for the vanishing route line feature and is only applicable if an instance of VanishingRouteLine was supplied at the time of instantiation.

Return

a state representing the side effects to be rendered on the map which will update the appearance of the route line or null if the vanishing route line feature is inactive.

Parameters

offset

a value representing the percentage of the distance traveled along the route
