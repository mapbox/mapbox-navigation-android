Edge

class Edge
Content copied to clipboard

Basic Edge class

Parameters

id

identifier of the directed edge (it's unique)

level

the level of the Edge (0 being the mpp, 1 branches of the mpp, 2 branches of level 1 branches, etc)

probability

the probability for this edge in percentage to be taken by the driver. The probabilities of all outgoing edges on a single intersection sum up to 1.

heading

heading when starting to move along the edge. The value is in degrees in the range [0, 360)

length

the Edge's length in meters

out

the outgoing Edges

parent

the parent Edge

functionRoadClass

the edge's FunctionalRoadClass

speed

average speed along the edge in m/s

ramp

is the edge a ramp?

motorway

is the edge a motorway?

bridge

is the edge a bridge?

tunnel

is the edge a tunnel?

toll

is the edge a toll road?

names

an array of road names

curvature

binned number denoting the curvature degree of the edge (0-15)

geometry

optional geometry if requested

speedLimit

max speed of the edge (speed limit) in m/s

laneCount

the number of lanes on the edge (does not change mid-edge)

meanElevation

mean elevation along the edge in meters

countryCode

ISO 3166-1 alpha-3 country code

stateCode

a state inside a country (ISO 3166-2)
