UpcomingRouteAlert

class UpcomingRouteAlert

Holds the route alert and the distance to the point where the alert occurs, or start an alert if it has length.

Parameters

routeAlert - route alert

distanceToStart - distance to the start of the alert. If the alert has a length, and we've passed the start point, this value will be negative until we cross the finish point of the alert's geometry. This negative value, together with RouteAlertGeometry.length can be used to calculate the distance since the start of an alert.
