RouteAlert

abstract class RouteAlert

Abstract class that serves as a base for all route alerts.

Available alert types are:

  • TunnelEntranceAlert
  • CountryBorderCrossingAlert
  • TollCollectionAlert
  • RestStopAlert
  • RestrictedAreaAlert

Parameters

alertType - constant describing the alert type, see RouteAlertType.

coordinate - location of the alert or its start point in case it has a length

distance - distance to this alert since the start of the route

alertGeometry - optional geometry details of the alert if it has a length
