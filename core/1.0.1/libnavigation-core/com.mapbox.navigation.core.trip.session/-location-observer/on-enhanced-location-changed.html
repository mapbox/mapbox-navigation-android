onEnhancedLocationChanged

abstract fun onEnhancedLocationChanged(enhancedLocation: Location, keyPoints: List<Location>): Unit

Provides the best possible location update, snapped to the route or map-matched to the road if possible.

Parameters

enhancedLocation - the best possible location update

keyPoints - a list (can be empty) of predicted location points leading up to the target update. The last point on the list (if not empty) is always equal to enhancedLocation.
