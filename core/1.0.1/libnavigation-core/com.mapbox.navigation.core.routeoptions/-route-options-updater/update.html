abstract fun update(routeOptions: RouteOptions?, routeProgress: RouteProgress?, location: Location?): RouteOptionsResult

Provides a new RouteOptions instance based on the original request options and the current route progress.

Returns null if a new RouteOptions instance cannot be combined based on the input given. When null is returned new route is not fetched.
