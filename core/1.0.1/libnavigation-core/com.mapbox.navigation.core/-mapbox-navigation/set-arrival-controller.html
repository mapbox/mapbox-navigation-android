setArrivalController

@JvmOverloads fun setArrivalController(arrivalController: ArrivalController? = AutoArrivalController()): Unit

Set your own controller to determine when drivers arrived at stops via ArrivalController. Use navigateNextRouteLeg to manually move navigator to the next stop. To reset to the automatic arrival controller, call setArrivalController.

By default uses AutoArrivalController.

Set null disables arrival at stops completely. Use this if you want to write your own mechanism for handling arrival at stops.
