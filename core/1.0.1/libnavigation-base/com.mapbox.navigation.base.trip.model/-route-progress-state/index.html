RouteProgressState

enum class RouteProgressState

Contains the various progress states that can occur while navigating.

Enum Values

ROUTE_INVALID

The com.mapbox.api.directions.v5.models.DirectionsRoute provided via com.mapbox.navigation.core.MapboxNavigation.setRoutes when in Active Guidance is not valid.

ROUTE_INITIALIZED

The DirectionsRoute is valid and com.mapbox.navigation.core.MapboxNavigation is waiting for sufficient android.location.Location updates from the com.mapbox.android.core.location.LocationEngine.

ROUTE_COMPLETE

The user has arrived at the destination of the given com.mapbox.api.directions.v5.models.RouteLeg.

LOCATION_TRACKING

com.mapbox.navigation.core.MapboxNavigation is now confidently tracking the location updates and processing them against the route.

LOCATION_STALE

A lack of android.location.Location updates has caused a lack of confidence in the progress updates being sent.

OFF_ROUTE

State when we detect an off-route.

ROUTE_UNCERTAIN

State when we start following a route.
