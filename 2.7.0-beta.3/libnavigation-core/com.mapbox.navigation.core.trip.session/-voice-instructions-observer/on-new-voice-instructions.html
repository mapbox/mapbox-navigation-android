onNewVoiceInstructions

abstract fun onNewVoiceInstructions(voiceInstructions: VoiceInstructions)
Content copied to clipboard

Called every time on a new VoiceInstructions is applicable. Only new voice instructions are available via this observer because the timing of delivering and playing out an instruction is critical for correct navigation experience. See {@link com.mapbox.navigation.base.trip.model.RouteProgress.voiceInstructions} to get a current voice instruction.
