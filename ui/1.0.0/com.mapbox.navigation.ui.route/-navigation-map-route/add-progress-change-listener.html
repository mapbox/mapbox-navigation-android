addProgressChangeListener

open fun addProgressChangeListener(@NonNull navigation: MapboxNavigation): Unit

This method will allow this class to listen to route progress and adapt the route line whenever TripSessionState#STARTED. In order to use the vanishing route line feature be sure to enable the feature before calling this method.
