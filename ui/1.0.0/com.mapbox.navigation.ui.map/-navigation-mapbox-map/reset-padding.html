resetPadding

open fun resetPadding(): Unit

This method resets the map padding to the default padding that is generated when navigation begins (location icon moved to lower half of the screen) or the custom padding that was last passed via #adjustLocationIconWith(int[]).

The custom padding will be used if it exists, otherwise the default will be used.
