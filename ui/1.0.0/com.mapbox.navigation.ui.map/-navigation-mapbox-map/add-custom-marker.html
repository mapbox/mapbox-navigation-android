addCustomMarker

open fun addCustomMarker(options: SymbolOptions!): Symbol!

Adds a custom marker to the map based on the options provided.

Please note, the map will manage all markers added. Calling NavigationMapboxMap#clearMarkers() will clear all destination / custom markers that have been added to the map.

Parameters

options - SymbolOptions!: for the custom com.mapbox.mapboxsdk.plugins.annotation.Symbol

Return
Symbol!: the com.mapbox.mapboxsdk.plugins.annotation.Symbol that added to the map or null if adding fails
