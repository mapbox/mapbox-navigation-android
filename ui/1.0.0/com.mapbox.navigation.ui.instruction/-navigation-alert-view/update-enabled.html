updateEnabled

open fun updateEnabled(isEnabled: Boolean): Unit

This method enables or disables the alert view from being shown during off-route events.

Note this will only happen automatically in the context of the NavigationView or a NavigationViewModel has been added to the instruction view with InstructionView#subscribe(LifecycleOwner, NavigationViewModel).

Parameters

isEnabled - Boolean: true to show during off-route events, false to hide
