updateBannerInstructionsWith

open fun updateBannerInstructionsWith(instructions: BannerInstructions!): Unit

Use this method inside a BannerInstructionsObserver to update banner instruction view.

This includes the maneuverView, banner text and sub step information. A default ManeuverModifier#RIGHT driving side will be used if driving side is unavailable.

Parameters

instructions - BannerInstructions!: for banner info used to populate the views
