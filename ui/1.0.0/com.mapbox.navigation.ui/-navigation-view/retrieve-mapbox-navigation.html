retrieveMapboxNavigation

@Nullable open fun retrieveMapboxNavigation(): MapboxNavigation?

Returns the instance of MapboxNavigation powering the NavigationView once navigation has started. Will return null if navigation has not been started with NavigationView#startNavigation(NavigationViewOptions).

Return
MapboxNavigation?: mapbox navigation, or null if navigation has not started
