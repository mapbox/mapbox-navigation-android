resetCameraPositionWith

open fun resetCameraPositionWith(trackingMode: Int): Unit

Resets the map camera / padding to the last known camera position.

You can also specify a tracking mode to reset with. For example if you would like to reset the camera and continue tracking, you would use NavigationCamera#NAVIGATION_TRACKING_MODE_GPS.

Parameters

trackingMode - Int: the tracking mode
