OnWayNameChangedListener

interface OnWayNameChangedListener

A listener that can be added to the NavigationMapboxMap with NavigationMapboxMap#addOnWayNameChangedListener(OnWayNameChangedListener).

This listener is triggered when a new way name is found along the route and it only triggers after NavigationMapboxMap#updateWaynameQueryMap(boolean) fires. This listener will be triggered regardless of the map way name visibility. (NavigationView#setWayNameActive(boolean)). This is so you can hide our implementation of the way name UI and update your own if you'd like.
