onNewRouteProgress

open fun onNewRouteProgress(routeProgress: RouteProgress!): Unit

Can be used to manually update the route progress.

NavigationMapboxMap automatically listens to RouteProgressObserver#onRouteProgressChanged(RouteProgress) when a progress observer is subscribed with #addProgressChangeListener(MapboxNavigation) and invoking this method in that scenario will lead to competing updates.

Parameters

routeProgress - RouteProgress!: current progress
