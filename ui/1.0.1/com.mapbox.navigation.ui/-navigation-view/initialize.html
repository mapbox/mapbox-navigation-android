open fun initialize(onNavigationReadyCallback: OnNavigationReadyCallback!, @NonNull initialMapCameraPosition: CameraPosition): Unit

Should be called after NavigationView#onCreate(Bundle).

This method adds the OnNavigationReadyCallback, which will fire the ready events for this view.

This method also accepts a CameraPosition that will be set as soon as the map is ready. Note, this position is ignored during a configuration change in favor of the last known map position.

Parameters

onNavigationReadyCallback - OnNavigationReadyCallback!: to be set to this view

initialMapCameraPosition - CameraPosition: to be shown once the map is ready
