toggleMute

open fun toggleMute(): Unit

This method toggles the mute state of speech player. It also updates the UI to reflect the new state of speech player.

Should be called after NavigationView#startNavigation. Otherwise, it does nothing.

Can check the current mute state by calling NavigationView#isVoiceGuidanceMuted()
