retrieveNavigationMapboxMap

@Nullable open fun retrieveNavigationMapboxMap(): NavigationMapboxMap?

Gives the ability to manipulate the map directly for anything that might not currently be supported. This returns null until the view is initialized.

The NavigationMapboxMap gives direct access to the map UI (location marker, route, etc.).

Return
NavigationMapboxMap?: navigation mapbox map object, or null if view has not been initialized
