RecenterButton

open class RecenterButton : ConstraintLayout, NavigationButton

Button used to re-activate following user location during navigation.

If a user scrolls the map while navigating, the com.mapbox.navigation.ui.summary.SummaryBottomSheet is set to hidden and this button is shown.

This button uses a custom TranslateAnimation with OvershootInterpolator to be shown.
