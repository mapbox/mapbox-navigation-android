showAlternativeRoutes

open fun showAlternativeRoutes(alternativesVisible: Boolean): Unit

Toggle whether or not you'd like the map to display the alternative routes. This options great for when the user actually begins the navigation session and alternative routes aren't needed anymore.

Parameters

alternativesVisible - Boolean: true if you'd like alternative routes to be displayed on the map, else false
