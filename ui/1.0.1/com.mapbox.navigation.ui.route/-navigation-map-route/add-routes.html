addRoutes

open fun addRoutes(@NonNull directionsRoutes: MutableList<out DirectionsRoute!>): Unit

Provide a list of DirectionsRoutes, the primary route will default to the first route in the directions route list. All other routes in the list will be drawn on the map using the alternative route style.

Parameters

directionsRoutes - MutableList<out DirectionsRoute!>: a list of direction routes, first one being the primary and the rest of the routes are considered alternatives.
