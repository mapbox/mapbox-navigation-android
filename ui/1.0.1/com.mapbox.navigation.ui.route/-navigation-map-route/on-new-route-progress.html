onNewRouteProgress

open fun onNewRouteProgress(@NonNull routeProgress: RouteProgress): Unit

Can be used to manually update the route progress.

NavigationMapRoute automatically listens to RouteProgressObserver#onRouteProgressChanged(RouteProgress) when a progress observer is subscribed with #addProgressChangeListener(MapboxNavigation, boolean) and invoking this method in that scenario will lead to competing updates.

Parameters

routeProgress - RouteProgress: current progress
