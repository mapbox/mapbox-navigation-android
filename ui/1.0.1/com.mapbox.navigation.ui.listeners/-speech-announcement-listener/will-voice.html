willVoice

@NonNull abstract fun willVoice(announcement: VoiceInstructions!): VoiceInstructions

Listener tied to voice announcements that are about to be voiced.

To prevent the given announcement from being announced, you can return null and it will be ignored.

If the SpeechAnnouncement#ssmlAnnouncement is malformed, the com.mapbox.navigation.ui.voice.SpeechPlayer will fall back to what is included in the SpeechAnnouncement#announcement with android.speech.tts.TextToSpeech.

Parameters

announcement - VoiceInstructions!: about to be announced

Return
VoiceInstructions: speech announcement to be played; null if should be ignored
