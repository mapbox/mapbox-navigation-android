setIsFallbackAlwaysEnabled

open fun setIsFallbackAlwaysEnabled(isFallbackAlwaysEnabled: Boolean): Unit

Set false to not fallback to TTS for voice guidance when the connection is slow. The default setting is enabled. The TTS is used in two cases: 1. when request polly voice fails 2. when connection is slow This setting only impact the 2nd case.

Parameters

isFallbackAlwaysEnabled - Boolean: true to use TTS when connection is slow, false otherwise
