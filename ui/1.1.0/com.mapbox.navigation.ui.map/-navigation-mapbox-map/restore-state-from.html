restoreStateFrom

open fun restoreStateFrom(@NonNull inState: Bundle): Unit

Can be used to restore a NavigationMapboxMap after it has been initialized.

This cannot be called in androidx.fragment.app.Fragment#onViewStateRestored(Bundle) because we cannot guarantee the map is re-initialized at that point.

Parameters

inState - Bundle: to store state variables
