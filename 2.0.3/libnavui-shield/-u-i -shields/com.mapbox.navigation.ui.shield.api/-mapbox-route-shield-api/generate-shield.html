fun generateShield(imageBaseUrl: String?, userId: String?, styleId: String?, mapboxShield: MapboxShield?, consumer: MapboxNavigationConsumer<Expected<RouteShieldError, RouteShield>>)
Content copied to clipboard

The function smartly uses BannerComponents.mapboxShield to request a route shield if present. If the network request returns an error it checks for BannerComponents.imageBaseUrl to request a shield. If neither BannerComponents.mapboxShield or BannerComponents.imageBaseUrl is available, the function returns a default blank shield wrapped in RouteShield or RouteShieldError otherwise.

a shield icon url
