Listener tied to voice announcements that are about to be voiced.

To prevent the given announcement from being announced, you can return nulland it will be ignored.

If the {@code SpeechAnnouncement#ssmlAnnouncement} is malformed,the com.mapbox.navigation.ui.voice.SpeechPlayer will fall back to what isincluded in the {@code SpeechAnnouncement#announcement} with android.speech.tts.TextToSpeech.

Return

speech announcement to be played; null if should be ignored

Parameters

announcement

about to be announced
